WEB DESIGN
OVERVIEW:

Web design is similar (in a very simplistic way) to traditional print publishing. Every website is an information display container, just as a book; and every Web page is like the page in a book. However, web design uses a framework based on digital code and display technology to construct and maintain an environment to distribute information in multiple formats. Taken to its fullest potential, web design is undoubtedly the most sophisticated and increasingly complex method to support communication in todayâ€™s world.

  • Participants can participate either in group or an individual
  • Maximum number of participants is 3
  • Time duration: 3 hours
  • Designing can be done by using HTML/Flash/Photoshop/PHP
  • Should support Chrome/Mozilla/Internet explorer
  • All softwares and images are provided by the committee itself
